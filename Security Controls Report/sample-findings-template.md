# Sample Findings Template

## Finding Title

Example: Unnecessary service enabled on Windows Server

## Severity

Low / Medium / High

## Description

Describe what was observed and why it matters.

## Risk

Explain the operational or security risk.

## Recommendation

Provide a practical remediation step.

## Validation

Explain how to confirm the issue was fixed.

## Control Mapping

Example: NIST SP 800-53 CM-6, SC-7, AU-2

## Status

Open / In Progress / Remediated / Accepted Risk
