# NIST SP 800-53 Control Mapping

This document maps project activities to selected NIST SP 800-53 control families.

| Control Family | Example Control | Project Activity |
|---|---|---|
| AC - Access Control | AC-2 Account Management | Review user and administrator access |
| AC - Access Control | AC-3 Access Enforcement | Restrict access through groups and firewall rules |
| AU - Audit and Accountability | AU-2 Audit Events | Enable audit policy for logon, account, and system events |
| AU - Audit and Accountability | AU-6 Audit Review | Review logs and document security-relevant events |
| CM - Configuration Management | CM-2 Baseline Configuration | Define hardened server baseline |
| CM - Configuration Management | CM-6 Configuration Settings | Apply security configuration settings |
| IA - Identification and Authentication | IA-5 Authenticator Management | Enforce password and lockout policy review |
| SC - System and Communications Protection | SC-7 Boundary Protection | Review firewall rules and network exposure |
| SI - System and Information Integrity | SI-2 Flaw Remediation | Review patching and update posture |
| SI - System and Information Integrity | SI-4 System Monitoring | Validate logs and monitoring after changes |

## Notes

This mapping is not a full compliance assessment. It is a project-level reference showing how technical hardening activities align with recognized security control families.
